В папке с установленной игрой замените файл `CBM_win/CBM-win32-x64/resources/app.asar`
на русифицированный: [скачать здесь](https://drive.google.com/file/d/1b8l1ozhcTrMoeVQn_hMIPBHfE3HSa0Qy/view?usp=sharing).
